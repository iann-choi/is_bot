# 접근제어 시스템 계정 신청 방법
## 1. 접근제어 시스템 계정 신청서 작성
  - 아마란스 접속
  - 전자결재
  - 결재작성
  - '사용자 계정 신청서' 작성
    - 제목 : '접근제어 시스템 계정 신청' 으로 작성해 주세요
    - 소속구분 : 임직원일 경우 '임직원', 외부자일 경우 '외부자'
    - 관리 담당자 : 외부자일 경우 해당 외부자의 업무 관리하는 담당자(aitrics 임직원)
    - 소속 : 소속 부서
    - 성명 : 신청자 성명
    - 사용기간 : 외부자이거나 단기 사용자의 경우 기재해 주세요
    - 신청 구분 : '생성', '변경', '삭제' 중 선택해 주세요
    - 계정 권한 : '관리자 권한', '일반사용자 권한' 중 선택해 주세요
    - 신청 ID : 회사 메일 계정으로 작성해 주세요
    - 업무 목적 : 업무 목적 작성해 주세요
    - 대상 시스템 : 접속이 필요한 서버, DB명 확인 후 작성해 주세요
    - 비고 : 추가 요청사항 또는 알림이 필요한 정보가 있다면 기재해 주세요
  - 결재 상신
  - 결재 완료 후 결재문서의 담당자 댓글 확인
    - 예: 계정 생성 완료되었습니다.
## 2. 서버 계정 생성
  - 대상 서버 사용자 계정 생성
## 3. 접근제어 시스템 에이전트 설치 및 접속 확인
  - '접근제어 시스템 적용 매뉴얼' 참고하여 에이전트 설치
